Olympic Medals API

NOTE: this documentation exists for posterity, but as the sochi2014 website is no longer active, this application is also non-functioning. It remains here for posterity. See the original blog post for more information.

Olympic Medals API publishes a JSON payload containing the current medal count for the Sochi 2014 Olympic Games. The full source code for this API is available on GitHub

Country Medals 

Country Medal Count
GET
/api/v1/medals/{country_id}

Return current medal count for a single country, based on country_id shortname.

For example, http://olympics.clearlytech.com/api/v1/medals/united-states
